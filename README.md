# Projeto Estágio Unimed

### Primeiros passos:
Para executar basta ter o nodejs 16+ instalado.
> npm update <br>
> npm install <br>
> node index.js <br>

Obs..: Se faz necessário editar o arquivo db.json com os dados de um banco de dados MySql.

### Feito utilizando:

- NodeJs
- ExpressJs
- HTML
- Bootstrap
- Javascript
- Banco de Dados MySql
